# Epsilon Kernel 
## This is my hobby project

## How to Compile?
you need i686-elf cross compiler, and also need QEMU.

'make' --> creating some files and others
'make clean' --> remove files (.bin and .o )
'make run' --> starting boot and you see the BOOT OK!

## about Kernel

The kernel will be monolithic because, no matter how old it may be, it is very practical and my preferred choice in terms of performance. 
This kernel will be operated via the terminal, meaning there will be no desktop applications. However, the source code is open, 
so anyone who wishes to can use this code to develop desktop applications specifically for this kernel type.

The reason I’m undertaking this project is to have a custom and practical kernel that I can use myself. 
I want it to be suitable for scientific, academic, and everyday use. Now? It’s time to develop!
